HTML Cheat Sheet

One of the first things we learn in web development is HTML, which stands for HyperText Markup Language and is used to define the structure and content of web pages. Although it is not considered a programming language, it requires handling many tags, attributes, and properties.

Remembering all this information can be challenging, which is why having a cheat sheet can be really helpful for beginners and even more experienced programmers.

Below is a Cheat Sheet with some basic HTML elements:

HTML Tag Description Example
<h1> to <h6> Headings <h1>Main Heading</h1>
<p> Paragraph <p>This is a paragraph.</p>
<br> Line break <br>
<hr> Horizontal rule <hr>
<strong> Bold text <strong>Bold text</strong>
<em> Italic text <em>Italic text</em>
<a> Anchor (link) <a href="https://example.com">Link to Example</a>
<img> Image <img src="image.jpg" alt="Image description">
<ul> Unordered list <ul><li>Item 1</li><li>Item 2</li></ul>
<ol> Ordered list <ol><li>Item 1</li><li>Item 2</li></ol>
<table> Table <table><tr><th>Header 1</th><th>Header 2</th></tr><tr><td>Cell 1</td><td>Cell 2</td></tr></table>
<tr> Table row Used inside <table> to create a row.
<th> Table header Used inside <tr> for a header cell.
<td> Table data Used inside <tr> for a data cell.
<input> Form input field <input type="text" placeholder="Enter text">
<label> Label for form input <label for="name">Name:</label><input id="name" type="text">
