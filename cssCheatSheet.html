CSS Cheat Sheet

Just like HTML, CSS is also not considered a programming language, but it is essential for web development. CSS stands for Cascading Style Sheets and is responsible for defining the visual presentation of a webpage. It controls the layout, design, and appearance of HTML elements, such as text, colors, spacing, and positioning.

CSS allows for responsive design, animation, and user interface customization, enhancing both the aesthetic and functional aspects of a website. CSS can be more challenging to master than HTML, and for this reason, a cheat sheet can be a very helpful tool for programmers in their day-to-day work.

Below is a cheat sheet with some basic CSS elements:

CSS Property Description Example
color Sets the color of text or elements color: blue;
background-color Sets the background color of an element background-color: lightgray;
font-family Sets the font family for text font-family: Arial, sans-serif;
font-size Sets the font size font-size: 16px;
font-weight Sets the font weight (boldness) font-weight: bold;
text-align Sets the text alignment text-align: center;
margin Sets the margin around an element margin: 10px;
padding Sets the padding inside an element padding: 10px;
border Sets the border of an element border: 1px solid black;
width Sets the width of an element width: 200px;
height Sets the height of an element height: 100px;
display Sets the display type of an element display: block; or display: inline-block;
flex Sets the flexibility of an item in a flex container flex: 1;
word-spacing Sets the space between words word-spacing: 5px;
text-decoration Sets text decorations like underline, overline, or line-through text-decoration: underline;
